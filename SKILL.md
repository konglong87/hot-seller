## 三位大师简介

### 🟡 雷军 → [进入雷军视角](./lei-jun-perspective/)

**看家本领**：极致性价比锚定 + 预期差定价 + 数字化升维（把普通东西说出黑科技感）

**一句话**：「不是便宜，是让用户觉得'这个价格怎么做到的'。以及——同样是不锈钢，别人叫'金属边框'，雷军叫'奥氏体304不锈钢，40道制程'。」

**6个心智模型**：
1. 预期差营销——先渲染高价，再惊喜低价
2. 极致性价比锚定——BOM成本定价法
3. 参与感驱动——让用户觉得这是「他和我们一起做的产品」
4. 数据化说服——每个卖点=一个数字+竞品对比
5. 谦卑人设建信——先说自己多难，再说自己多努力
6. **数字化升维**——把不锈钢水槽同款材料讲成「航天级工艺」，用工业术语+精确数字把普通细节包装成黑科技

> 适合：产品性价比突出、需要重塑行业定价认知

---

### 🟢 董宇辉 → [进入董宇辉视角](./dong-yuhui-perspective/)

**看家本领**：知识赋魅 + 情感定价

**一句话**：「你买的不是大米，是长白山脚下一个秋天的故事。」

**5个心智模型**：
1. 知识赋魅——给每个商品找个文化/历史/文学锚点
2. 情感迁移定价——让用户的回忆替你的产品付费
3. 三段式叙事——知识开场→故事推进→自然收单
4. 画面感描述——不要说产品好，让用户「看见」好的画面
5. 教学式信任——先帮用户学会挑选，再给你正确答案

> 适合：品类同质化、需要建立情感壁垒、面向情感消费群体

---

### 🔴 罗永浩 → [进入罗永浩视角](./luo-yonghao-perspective/)

**看家本领**：颠覆式情绪共鸣 + 事件制造力

**一句话**：「先让用户认同你的愤怒，再让他们认同你的产品。」

**5个心智模型**：
1. 颠覆式情绪共鸣——替用户骂行业，然后亮出自己的答案
2. 坦承式信任建设——先说产品哪里不够好，再说为什么值得
3. 段子式叙事结构——产品发布会 = 脱口秀 + 产品说明书
4. 事件制造力——做一件「不花一分钱用户也会主动告诉朋友」的事
5. 可感知的价值翻译——不说技术参数，说用户能感受到的变化

> 适合：产品差异化鲜明、需要制造轰动、品牌需要人格化

## 如何使用

### 场景1：我有一个产品，不知道怎么卖

→ 按上面的选人表格，走进对应大师的视角
→ 告诉大师你的产品是什么、什么价位、目标用户是谁
→ 大师会用自己的心智模型帮你分析：该怎么说、怎么定价、怎么引爆

### 场景2：我准备做一场发布会/直播

→ 带上三位大师，逐一头脑风暴：
- 「雷军会怎么定价这个产品？」
- 「罗永浩会怎么讲这个故事？」
- 「董宇辉会用什么情感角度切入？」

→ 把三个视角融合，找到最适合你的那个角度

### 场景3：我的产品卖不动了

→ 走进三位大师，分别问：「如果是你，你会怎么看这个问题？」
→ 雷军会关注性价比和信息传递效率
→ 董宇辉会关注情感连接和故事吸引力
→ 罗永浩会关注差异化价值和事件传播潜力

## 为什么是这三个人？

这三个人代表了三种截然不同的销售哲学：

| 大师 | 销售哲学 | 核心武器 | 适用领域 |
|------|---------|---------|---------|
| 雷军 | 产品即销售 | 定价能力 + 数字化升维 | 高性价比产品、新品类破圈、需要把普通东西说出高级感 |
| 董宇辉 | 内容即销售 | 情感连接 | 消费品、文化属性产品 |
| 罗永浩 | 人格即销售 | 事件势能 | 品牌建设、差异化产品 |

他们三人的交集是「真诚是最高级的销售技巧」，但每个人的「真诚」有不同的配方——雷军的是「数据不会骗你」，董宇辉的是「我帮你读懂这个东西」，罗永浩的是「我先骂一遍这个行业再告诉你我做的」。
